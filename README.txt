小蔡同学文档工作室 - 美观后台增强版

一、项目内容
1. public/index.html：前台接单网站。
2. public/admin.html：美观后台管理页面。
3. server.js：Node.js + Express 后端。
4. package.json：运行配置。

二、后台入口
部署成功后，在网站地址后面加：
/admin

例如：
https://你的项目名.onrender.com/admin

三、默认后台密码
123456

正式使用建议在 Render 设置环境变量：
ADMIN_PASSWORD=你自己的密码

四、后台功能
1. 密码登录。
2. 订单统计卡片。
3. 搜索订单。
4. 按服务类型筛选。
5. 按订单状态筛选。
6. 按时间或金额排序。
7. 查看订单详情弹窗。
8. 编辑客户称呼、联系方式、服务类型、截止时间、需求内容。
9. 修改订单状态：新订单、已联系、已报价、已收款、制作中、已完成、已取消。
10. 记录订单金额。
11. 写自己的内部备注。
12. 删除订单。
13. 导出 CSV 表格。
14. 手机端适配。

五、Render 部署更新方法
1. 解压本压缩包。
2. 进入 GitHub 仓库。
3. 删除或覆盖原来的 public 文件夹、server.js、package.json。
4. 上传本版本对应文件。
5. Commit changes。
6. Render 会自动重新部署。
7. 部署完成后打开 /admin。

六、重要说明
当前订单保存在 orders.json 文件里，适合前期测试和小规模接单。
Render 免费服务重启后文件可能丢失。
后期如果要长期稳定使用，建议升级为数据库版本。
